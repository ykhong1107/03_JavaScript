복합 대입 연산자(+=, -=, *=, /*, %=)

    기존 변수에 저장되어 있는 값에
    새롭게 대입되는 값의 산술 결과를 대입하는 연산자

    - 코드가 짧고, 속도가 우위를 가지고 있음

    ex)
    let tmep = 10;

    temp += 5; // temp = temp + 5;

논리 연산자(&&, ||, ! )

    && (AND) 연산자 -> A && B

    - A와 B 모두 true 일 경우에만
      A && B 결과가 true인 연산자 (나머진 false)
    
    - 글/말/문장에서
      ~와, 그리고(이고), 이면서(면서), ~까지, ~부터, ~사이
      라는 글귀가 있으면 AND 연산을 작성

    ex) 104는 100 이상의 숫자 이면서 짝수인가?
        ->    (104 >= 100)   &&    (104  %  2 == 0) -> true
  
    || (OR 연산자) -> A || B

    - A와 B 모두 false 일 경우에만 false (나머진 true);

    - 문장에
      또는, ~이거나(거나)
      글귀가 있으면 OR 연산을 수행

    ex) 4는 10을 초과하거나 짝수인가?
      ->  (4 > 10)  ||  (4 % 2 === 0 )
  
    !(NOT) 연산
    - 논리 값을 반대로 바꾸는 연산자

    !true -> false
    !false -> true

    ex)
    !(true && false) -> true

    (10 > 5) || !(10 % 2 === 0) -> true

    !!!true -> false
